# 🦆 날아라 오리야!

> 4색 오리를 격추하라! 놓치면 강아지가 독한 멘트를 날리는 웨이브 슈팅 게임!

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/Duck-hunt](https://mindweaver74.github.io/Duck-hunt)

---

## 🎮 게임 방법

1. 화면을 가로질러 날아다니는 오리를 클릭/탭해서 격추하세요
2. 제한 시간 안에 격추하지 못하면 강아지가 나타나 놀립니다
3. 웨이브가 올라갈수록 오리가 더 빠르게 날아다닙니다
4. 최대한 많은 오리를 격추해 고득점을 노리세요!

---

## 🕹️ 조작법

| 환경 | 조작 |
|------|------|
| PC | 마우스 클릭 |
| 모바일 | 화면 탭 |

---

## ✨ 주요 특징

- 🦆 **4색 오리** — 색상마다 점수가 다름
- 😤 **강아지 리액션** — 놓칠 때마다 독한 멘트 12종 랜덤
- 🌊 **웨이브 시스템** — 웨이브마다 오리 속도와 수 증가
- 🎯 **정확도 기록** — 발사 수 대비 명중률 표시

---

## 💡 고득점 전략

- 오리가 방향을 바꾸기 직전 타이밍에 클릭하면 맞추기 쉬워요
- 화면 가장자리로 향할 때 미리 앞쪽을 클릭하세요
- 점수가 높은 색상 오리를 우선 격추하세요
