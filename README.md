# 🦆 덕헌트 REBORN

> 날아다니는 오리를 잡으세요! 3번 놓치면 강아지가 놀려줍니다 🐕

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/duckhunt-reborn](https://mindweaver74.github.io/duckhunt-reborn)

---

## 🎮 게임 방법

1. 화면을 **터치/클릭**해서 날아다니는 오리를 잡으세요
2. 오리를 놓치면 강아지가 말풍선으로 놀려줍니다 😂
3. **3번 놓치면 게임 오버**
4. 모든 오리를 잡으면 다음 웨이브로 진행
5. 웨이브가 올라갈수록 오리가 많아지고 빨라집니다

---

## 🏆 점수 체계

| 상황 | 점수 |
|------|------|
| 오리 1마리 처치 | 100점 + 웨이브 보너스 |
| 2콤보 | 점수 x2 |
| 3콤보 | 점수 x3 |
| 최대 5콤보 | 점수 x5 |

- 연속으로 잡을수록 콤보 배율 상승
- 한 번이라도 놓치면 콤보 초기화

## ✨ 특징

- 🎨 **자연 배경** — 하늘, 구름, 풀밭, 나무 Canvas 직접 렌더링
- 🐕 **강아지 캐릭터** — 꼬리 흔들기, 놀릴 때 웃음 애니메이션
- 💬 **Canvas 말풍선** — 강아지 입에서 직접 나오는 말풍선
- 🪶 **깃털 파티클** — 오리 잡을 때 깃털이 흩날림
- 🌊 **웨이브 시스템** — 클리어할수록 난이도 증가
- 🎵 **효과음** — 총소리/명중/콤보/웨이브클리어/강아지 웃음
- 🏆 **최고 기록 저장** — 브라우저에 자동 저장
